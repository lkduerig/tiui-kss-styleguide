1.2.1Brand Colour Spectrum #permalink

The main color in Tieto is blue. There are three complimentary colors for use for different purposes. Usually every application have one main color, others are used for contrast for special elements. The larger squares (light, medium, dark) are defined by the Brand Guidelines. Smaller (very light, lightly dark) are defined by the UX guidelines, for having more possibilities for fine tuning subtle design decissions. 1.6Typography #permalink

Default font family for interfaces is Arial. The body text color is #333 (or darkest gray color as described above). The font size is of course variable but for text paragraphs 13px font size is recommended. Line height is dependent on the font sizes but for body text is 21px (160% of font size).

For other purposes the Tieto brand font family Helvetica Neue can be used, but please keep in mind that it is not available on all computers. You can take use of existing Fonts.com license for serving Helvetica Neue as a web font. Please get in touch with Marketing contact persons for that.
