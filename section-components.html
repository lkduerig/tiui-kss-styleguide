2.4Icons #permalink

Pixicon icon set - Transparent PNGs & Endless scalable vector icons for User Interface Designers and Developers. Icons are available in icons in different sizes up to 128px .png, colors and formats including vectors. http://pixicon.jackietrananh.com/

Gentleface icon set (.zip, 90Mb) - Gentleface icon set provides icons in png format for sizes 16x16, 32x32 and 48x48 pixels. The full icon set includes .ico files and vector icons. We have purchased a royalty free license for commercial usage, but please read through the license agreement before using the icon set. Note: We avoid using gradients in icons, so in general gradient should be removed from icons before using them. http://www.gentleface.com

2.6Logos #permalink

Please refer to Brand Design Guidelines for more usage tips such as minimum logo size and choice between logos.

Examples
Default styling
.logo--primary-blue
Primary logotype, most commonly used and placed in the lower right corner. To be used on white background.
.logo--primary-white
Primary logotype, most commonly used and placed in the lower right corner. To be used on a coloured background.
.logo--primary-black
Primary logotype, most commonly used and placed in the lower right corner. Black is to be used on a white background and only in exceptional cases.
.logo--secondary-blue
Secondary logotype, used in top left corner. To be used on white background.
.logo--secondary-white
Secondary logotype, used in top left corner. To be used on a coloured background.
.logo--secondary-black
Secondary logotype, used in top left corner. Black is to be used on a white background and only in exceptional cases.
.logo--tieto-com--blue
Tieto.com text to be used on white background.
.logo--tieto-com--white
